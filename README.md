# Prototipo-La-Esperanza
Este proyecto es un prototipo de aplicación que se ejecuta en entorno local usando Node.js.
---
## Requisitos

Antes de comenzar, asegurarse de tener instalado:

* Node.js
* npm (incluido con Node.js)
* Visual Studio Code (recomendado)
---

## Ejecutar el proyecto de manera local

1. Abre la carpeta del proyecto en Visual Studio Code
2. Abre la terminal
3. Ejecuta el siguiente comando:
4. 
```bash
npm run dev
```

4. Después de ejecutar el comando, mostrara un enlace en la terminal (como el siguiente: http://localhost:5173/)
5. Abrir el enlace en el navegador para ver la aplicación
---

## Para ver la app en el celular

Para acceder desde el celular:

1. Asegurarse de que la computadora y el celular estén conectados a la **misma red wifi**
2. Ejecutar el siguiente comando en la terminal:

```bash
npm run dev -- --host
```
3. La terminal mostrará una URL (normalmente con la IP local, ej: http://192.168.x.x:5173/)
4. Abre ese enlace desde el navegador del celular
